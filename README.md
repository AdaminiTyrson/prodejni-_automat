Simulace prodejního automatu napsaná v Pythonu. Projekt je zaměřený na návrh logiky systému, práci se stavy a ošetření chybových situací (např. nedostatek kreditů, vyprodané zboží, vrácení peněz).

## Funkce
- výběr produktu
- kontrola dostupnosti zboží
- práce s kreditem / platbou
- výdej produktu
- vrácení peněz (change)
- aktualizace zásob

## Technologie
- Python
- objektově orientované programování (OOP)
